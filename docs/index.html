How it works

01

Build context from source code

What does this mean?
Instead of manually writing test cases, Agent Chewy reads your actual service code to understand how the API behaves. It learns the business rules, response codes, and edge cases directly from the source.

How it works:
Point it at your service repository. Agent Chewy scans Java/TypeScript files, finds test data patterns, extracts validation logic, and builds a "context library"‚Äîa knowledge base about your API.

02

Describe your test in plain English

What does this mean?
No need to learn Cucumber syntax or write code. Just describe what you want to test like you're talking to a colleague. Agent Chewy understands your intent and generates the complete test.

How it works:
Your plain English is matched against the context library. It finds the right test data, expected response code, and generates proper assertions‚Äîall without calling OpenAI (fast & free!).

Self-healing context validation

What does this mean?
APIs change over time. Instead of manually updating tests, Agent Chewy can detect when your context library is out of sync with the real API and automatically fix it.

How it works:
Run the validator. It calls the real API with test data for each response code, compares actual vs expected behavior, and updates your context library if there's a mismatch.

Test Creation with
AI Recorder

The AI Recorder captures user actions inside a web application and automatically converts them into test scenarios. It helps the AI Test Engineer create reliable and repeatable automated tests without writing code manually.

üé¨ Start Recording

How it works

01

Recording Test Steps

Confirm the Recorder is in Recording mode
When you run the recording command, Playwright Codegen opens a browser window with the Inspector panel. The recorder is automatically in recording mode‚Äîyou'll see Playwright commands being generated in real-time as you interact with the application.

Interact with the application inside the browser window
Simply perform your test scenario normally: click buttons, fill forms, navigate pages. The Recorder automatically logs your actions and generates corresponding test steps in real time. You can see each action being captured in the Playwright Inspector panel.

üìå Example Command
$ ./fully-auto-record.sh "Checkout Flow" "Complete purchase" "https://example.com"
üé¨ Browser opens with Playwright Inspector...
üëâ Interact with your application
üëâ Watch commands generate in real-time
üëâ Close browser when done
02

Generating Code and Saving

To save a recorded scenario:
After you close the browser, the recording script automatically converts your actions into Cucumber/Gherkin feature files and TypeScript step definitions. The script saves your test with the name you provided when you ran the command.

Feature Name and Scenario Name
When you run ./fully-auto-record.sh "Test Name" "Description", the script uses your test name to generate the feature file (e.g., "chewy-test.feature"). The scenario name is automatically set to "Execute recorded actions", but you can edit the generated files afterward if needed. All files are saved to features/ui/ and src/steps/ui/ directories.

View your saved test
After the script completes, you can find your test files in the features directory. The generated files are ready to run immediately with Cucumber.

03

Playing Back a Scenario

Playback allows verification of test steps after saving
After generating and saving the test files, the fully-auto-record.sh script automatically runs the recorded steps to verify they work correctly. You'll see the test execution in real-time with pass/fail results.

To playback a scenario:
The script automatically executes the generated Cucumber tests using Playwright. You'll see each step being executed and whether it passes or fails. If tests fail, you can review the generated code and fix any issues, then run the tests again manually.

What is Agent Chewy?

Agent Chewy is an AI-powered test automation platform that revolutionizes how you create, maintain, and execute automated tests. It combines the power of AI with industry-standard testing frameworks (Cucumber/BDD, Playwright) to deliver fast, reliable, and maintainable test automation.

Unlike traditional test automation tools that lock you into proprietary formats, Agent Chewy generates standard Cucumber/BDD test code that you own completely. Your tests are stored in your Git repository, giving you full control and freedom to run them anywhere.
